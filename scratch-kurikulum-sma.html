VERIFIKASI REKURSI & FRAKTAL
  • Base case ada dan berfungsi — kedalaman 0 langsung berhenti
  • Tanpa base case → Scratch freeze atau error (siswa harus merasakan ini dulu!)
  • Kedalaman 1 = batang + 2 cabang pendek (terlihat benar)
  • Kedalaman 5+ = pohon yang memiliki struktur fraktal yang jelas
  • Parameter panjang berkurang di setiap level rekursi (× 0.7 atau sejenisnya)
  • Siswa bisa menjelaskan urutan eksekusi rekursi secara verbal (DFS order)
  • Siswa tahu: kenapa rekursi tanpa base case disebut "infinite recursion"?

# Terjemahan state machine musuh ke Python import math import random class Musuh: def __init__(self, x, y, level): self.x, self.y = x, y self.hp = level * 30 self.speed = level * 1.5 self.state = "PATROL" def jarak_ke(self, target_x, target_y): dx = target_x - self.x dy = target_y - self.y return math.sqrt(dx**2 + dy**2) def update_state(self, pemain_x, pemain_y): jarak = self.jarak_ke(pemain_x, pemain_y) if jarak < 30: self.state = "ATTACK" elif jarak < 120: self.state = "CHASE" else: self.state = "PATROL" def update(self, pemain_x, pemain_y): self.update_state(pemain_x, pemain_y) if self.state == "CHASE": dx = pemain_x - self.x dy = pemain_y - self.y dist = self.jarak_ke(pemain_x, pemain_y) if dist > 0: self.x += (dx/dist) * self.speed self.y += (dy/dist) * self.speed # Buat 5 musuh (setara "create clone" × 5 di Scratch) musuh_list = [Musuh(random.randint(-200,200), random.randint(-150,150), random.randint(1,5)) for _ in range(5)]
