SD — Level 1: Eksplorasi & Bermain

SEKOLAH DASAR · KELAS 4–6
Mengenalkan konsep dasar lewat visual, animasi, dan permainan sederhana
8 Modul
MODUL 01
Apa itu Scratch? Mengenal Antarmuka
  • Mengenal area Stage, Sprite, dan Blok
  • Drag and drop blok perintah
  • Menjalankan & menghentikan program
  • Memilih dan mengganti Sprite
🎯 Proyek: Sprite bergerak dan berkata "Halo!"
MODUL 02
Gerak & Animasi Dasar
  • Blok "gerak" — maju, mundur, putar
  • Ganti kostum untuk animasi berjalan
  • Blok "tunggu" untuk mengatur kecepatan
  • Gerakan memantul di tepi layar
🎯 Proyek: Animasi hewan berjalan
MODUL 03
Suara & Efek Visual
  • Memutar suara dari library
  • Rekam suara sendiri
  • Efek grafis (warna, ukuran, transparansi)
  • Ganti background cerita
🎯 Proyek: Cerita animasi bersound effect
MODUL 04
Kontrol Keyboard & Mouse
  • Blok "jika tombol ... ditekan"
  • Menggerakkan sprite dengan panah keyboard
  • Klik mouse untuk trigger aksi
  • Sprite mengikuti kursor
🎯 Proyek: Karakter yang bisa dikontrol pemain
MODUL 05
Pengenalan Kondisi (If-Then)
  • Konsep "jika ini terjadi, maka lakukan itu"
  • Blok "jika ... maka ..."
  • Deteksi menyentuh warna / sprite lain
  • Perbedaan aksi ya dan tidak
🎯 Proyek: Sprite bereaksi saat menyentuh benda
MODUL 06
Pengulangan (Loop Dasar)
  • Kenapa perlu pengulangan?
  • Blok "ulangi ... kali"
  • Blok "ulangi selamanya"
  • Animasi berulang otomatis
🎯 Proyek: Animasi hujan / bintang berkelip
MODUL 07
Variabel Sederhana & Skor
  • Apa itu variabel? (kotak penyimpan angka)
  • Membuat dan menampilkan variabel Skor
  • Menambah dan mengurangi nilai
  • Menampilkan nilai di layar
🎯 Proyek: Counter klik — klik sprite tambah poin
MODUL 08
Proyek Akhir SD: Game Mini
  • Menggabungkan semua konsep
  • Desain karakter & latar sendiri
  • Menambah kondisi menang/kalah
  • Presentasi game ke teman
🎯 Proyek: Game tangkap benda jatuh dari langit
🔧

SMP — Level 2: Logika & Struktur

SEKOLAH MENENGAH PERTAMA · KELAS 7–9
Membangun logika berpikir terstruktur, kondisi kompleks, dan proyek kolaboratif
8 Modul
MODUL 01
Review & Pendalaman Konsep Dasar
  • Recap blok gerak, suara, tampilan
  • Pengenalan mode Turbo & debugging
  • Membaca script yang sudah ada
  • Kebiasaan: komentar pada blok kode
🎯 Proyek: Analisis & perbaiki kode yang bermasalah
MODUL 02
Kondisi Kompleks (If-Else & And/Or)
  • Blok "jika ... maka ... jika tidak ..."
  • Operator logika: AND, OR, NOT
  • Kondisi bersarang (nested if)
  • Perbandingan nilai: < > =
🎯 Proyek: Kuis dengan pilihan benar/salah bertingkat
MODUL 03
Loop Lanjutan & Pola
  • Blok "ulangi sampai ..."
  • Loop bersarang untuk pola grid
  • Menggambar pola dengan pena
  • Spiral, segi-n, dan fraktal sederhana
🎯 Proyek: Menggambar pola geometri dengan kode
MODUL 04
Variabel, Input & Tipe Data
  • Variabel lokal vs global
  • Blok "tanyakan dan tunggu" untuk input user
  • Menyimpan jawaban di variabel
  • Operasi matematika pada variabel
🎯 Proyek: Kalkulator sederhana interaktif
MODUL 05
List (Array Dasar)
  • Konsep list — kumpulan data dalam satu tempat
  • Tambah, hapus, dan baca item dari list
  • Iterasi melalui list
  • Cek apakah item ada di list
🎯 Proyek: Daftar belanja / kamus kata sederhana
MODUL 06
Custom Blok (Fungsi/Prosedur)
  • Mengapa memecah kode jadi blok-blok kecil?
  • Membuat blok kustom sendiri
  • Blok dengan parameter input
  • Prinsip DRY: Don't Repeat Yourself
🎯 Proyek: Game dengan fungsi lompat, tembak, mati
MODUL 07
Pesan (Events & Broadcasting)
  • Komunikasi antar sprite via pesan
  • Broadcast & receive message
  • Koordinasi multi-sprite dengan timing
  • Membangun alur cerita interaktif
🎯 Proyek: Cerita interaktif dengan banyak karakter
MODUL 08
Proyek Akhir SMP: Game Lengkap
  • Desain sistem: level, nyawa, skor tinggi
  • Implementasi multiple screens (menu, game, game over)
  • Playtesting dan iterasi
  • Dokumentasi & presentasi proyek
🎯 Proyek: Platform game atau puzzle game 2D
⚙️

SMA — Level 3: Algoritma & Siap Python

SEKOLAH MENENGAH ATAS · KELAS 10–12
Memahami algoritma, struktur data, dan pola yang langsung paralel dengan Python
8 Modul
MODUL 01
Algoritma & Pseudocode
  • Apa itu algoritma? (langkah logis menyelesaikan masalah)
  • Menulis pseudocode sebelum koding
  • Flowchart → Scratch → nanti Python
  • Kompleksitas: solusi efisien vs tidak efisien
🎯 Proyek: Implementasi algoritma pencarian sederhana
MODUL 02
Struktur Data: List & Stack
  • List sebagai stack (LIFO) di Scratch
  • Push dan pop operasi manual
  • List sebagai queue (FIFO)
  • Aplikasi nyata: undo/redo, antrian
🎯 Proyek: Simulasi antrian kasir / undo system
MODUL 03
Algoritma Sorting & Searching
  • Bubble sort visual dengan Scratch
  • Linear search step-by-step
  • Visualisasi: melihat algoritma "bekerja"
  • Perbandingan jumlah langkah tiap metode
🎯 Proyek: Sorting visualizer animasi nyata
MODUL 04
Rekursi & Fraktal
  • Konsep "fungsi memanggil dirinya sendiri"
  • Rekursi dengan custom blok di Scratch
  • Menggambar pohon fraktal & Sierpinski
  • Base case dan stack overflow
🎯 Proyek: Pohon fraktal generatif dengan pena
MODUL 05
Simulasi & Pemodelan
  • Simulasi fisika: gravitasi, pantulan, gesekan
  • Random number untuk perilaku acak
  • Agen sederhana: flocking / boids
  • Cellular automata dasar (Game of Life)
🎯 Proyek: Simulasi bola biliar atau partikel
MODUL 06
Pola Desain: OOP Konseptual
  • Sprite sebagai "objek" dengan properti & perilaku
  • Kloning sprite = instansiasi objek
  • Pesan = method call antar objek
  • Bridging ke konsep class & object di Python
🎯 Proyek: Game dengan musuh ber-AI yang di-clone
MODUL 07
Scratch ↔ Python: Jembatan Transisi
  • Pemetaan blok Scratch → sintaks Python
  • if/else, for loop, while loop dalam Python
  • Fungsi di Python vs Custom Blok
  • List di Scratch vs List di Python
🎯 Proyek: Tulis ulang game Scratch sederhana ke Python
MODUL 08
Proyek Akhir SMA: Capstone Project
  • Desain proyek orisinal dari nol
  • Gunakan algoritma yang dipelajari
  • Dokumentasi teknis & user guide
  • Demo & pitching ke audiens
🎯 Proyek: AI sederhana / simulasi ilmiah / aplikasi edukatif
