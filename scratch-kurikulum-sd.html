Checklist Uji Coba — Modul 05
VERIFIKASI SEBELUM LANJUT KE MODUL BERIKUTNYA
  • Program berjalan normal saat TIDAK ada tabrakan
  • Saat sprite menyentuh musuh → reaksi yang benar terjadi (sound + pesan)
  • Saat sprite menyentuh item positif → reaksi yang berbeda (bukan game over)
  • Program berhenti (stop all) dengan benar setelah game over
  • Tidak ada "false positive" — reaksi terjadi hanya saat benar-benar menyentuh
  • Siswa bisa menjelaskan: "Apa yang terjadi jika kondisi TIDAK terpenuhi?"
⭐
Tantangan Bonus
  • Tambahkan kondisi "if touching edge" untuk membuat sprite memantul tanpa ke luar layar
  • Buat 3 jenis benda berbeda: musuh (game over), koin (+poin), obat (+nyawa)
  • Buat sprite berubah warna HANYA saat cursor mouse berada di atasnya

  • Siswa mampu mempresentasikan dan menjelaskan cara kerja gamenya sendiri
🎮
Spesifikasi Proyek: "Tangkap Buah!"
GAME ARCADE SEDERHANA · SATU PEMAIN
MEKANISME
Keranjang digerakkan kiri-kanan, tangkap buah jatuh
SKOR
+10 per buah ditangkap, -1 nyawa jika jatuh
KONDISI MENANG
Skor mencapai 100 poin
KONDISI KALAH
Nyawa habis (mulai dari 3 nyawa)
SPRITE

Catch sound, drop sound, win/lose music
RINCIAN PER PERTEMUAN

P-2 Koding Mekanik Utama — keranjang bergerak dengan keyboard, buah jatuh dengan kecepatan acak, deteksi tangkapan dan miss Game bisa dimainkan (belum ada win/lose)
P-3 Finishing & Presentasi — tambah screen menang/kalah, suara, efek, playtesting dengan teman, presentasi 2 menit ke kelas Game selesai + presentasi

Checklist Uji Coba Final — Proyek Akhir SD
SEMUA POIN HARUS TERPENUHI SEBELUM PRESENTASI
  • Reset: Klik bendera hijau → Skor = 0, Nyawa = 3, game mulai dari awal
  • Kontrol: Keranjang bergerak kiri-kanan dengan tombol panah, tidak keluar layar
  • Buah jatuh: Buah muncul di posisi X acak dan jatuh dari atas ke bawah
  • Tangkap: Saat buah menyentuh keranjang → Skor +10, suara catch, buah hilang
  • Miss: Saat buah sampai bawah tanpa ditangkap → Nyawa berkurang 1
  • Game Over: Nyawa = 0 → pesan game over + suara + program berhenti
  • Menang: Skor ≥ 100 → pesan menang + suara kemenangan
  • Tampilan: Skor dan Nyawa tampil di layar sepanjang permainan
